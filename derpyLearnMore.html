Derpy Derby

Besides a personal project of mine, Derpy Derby is the first multiplayer game I've developed. The game design was Paul Harden's, and I started development on it with him, Sam Levine, and Greg Beaudoin during a 2021 game jam hosted by Marist College. I did all the network programming, and I am continuing to work on it with Paul to release it in the near future. This was the first online multiplayer game made at Marist College.

I used Mirror for the network code and the addon LRM to host the game on an Ubuntu server I rent. There is a server list to easily join other hosted games and cars will spawn in opposite corners with different colors dependent on how many players are currently in the lobby. There are a few basic game modes right now and are currently being improved.
